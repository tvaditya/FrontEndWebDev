There are three html files here, the purpose is to understand the basic concpets of HTML5, CSS and Javascript
--------------------------------------------------------------
DatingWebsite.html - Show how to use and combine different HTML elements and use CSS to change some of their behaviours. It is basic for anyone to learn.
--------------------------------------------------------------
GuessTheColor.html - A simple game in which user has to guess the color. This is to introduce basic concepts of Javascript(programming logic,decisions, loops), as well as arrays and some math functions. Also to deal with mouse events.
--------------------------------------------------------------
FindTheFace.html - A simple game to find the extra face comparing to images. Understand the DOM strucuture and enhance event handling.
